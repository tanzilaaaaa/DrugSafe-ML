# User Guide

## Getting Started

### Accessing the Application
1. Start the Flask application: `python src/main.py`
2. Open your browser and navigate to `http://localhost:5001`
3. You'll see the Drug Interaction Checker interface

## Using the Drug Interaction Checker

### Single Drug Check
1. Enter a drug name in the search field
2. Click "Check Interactions"
3. View the interaction results and severity levels

### Batch Prediction
1. Enter multiple drug names (comma-separated)
2. Click "Batch Check"
3. Review all interactions at once

## Understanding Results

### Severity Levels
- **Low**: Minor interactions with minimal clinical significance
- **Moderate**: Interactions requiring monitoring
- **High**: Serious interactions requiring medical attention

## Features

- Real-time drug interaction checking
- Support for 15+ common medications
- Severity classification
- Batch processing capability
- Statistical insights
